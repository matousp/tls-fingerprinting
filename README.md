# TLS Fingerprinting
This repository contains datasets, scripts and papers related to TLS fingerprinting being researched by the [NES](https://www.fit.vut.cz/research/group/nes@fit/.en) research team at the [Brno University of Technology](https://www.fit.vut.cz/.en), Czech Republic. 

## Git folders
  * [datasets](datasets): Anotated datasets of desktop and mobile applications in form of PCAP and CSV files .
  * [tools](tools): Scripts for parsing PCAP files and extracting features into CSV files, scripts for computing statistics.
  * [papers](papers): Publications related to the TLS fingerprinting.

Details about this research can be found in the paper *Matoušek Petr, Ryšavý Ondřej a Burgetová Ivana. Towards Identification of Network Applications in Encrypted Traffic. In the Proceedings of the 8th Cyber Security in Networking Conference 2024, Paris, France, 2024*, see folder [papers](papers).
